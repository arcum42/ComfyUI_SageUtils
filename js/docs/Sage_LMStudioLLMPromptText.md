# Sage_LMStudioLLMPromptText

**LM Studio LLM Prompt (Text)**

Sends a text prompt to an LM Studio LLM and returns the result.

## Parameters
- **prompt**: The text prompt to send.
- **model**: The LM Studio model to use.

## Usage
Use to generate text completions or responses from an LM Studio LLM in your workflow.
