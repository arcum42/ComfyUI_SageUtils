# Sage_LMStudioLLMPromptVision

**LM Studio LLM Prompt (Vision)**

Sends a vision prompt to an LM Studio LLM and returns the result.

## Parameters
- **image**: The image to analyze.
- **prompt**: The text prompt to send.
- **model**: The LM Studio model to use.

## Usage
Use to generate vision-based completions or responses from an LM Studio LLM in your workflow.
