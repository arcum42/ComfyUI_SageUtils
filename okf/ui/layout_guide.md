This concept documents common layout patterns and container components used in Sage Utils UI.

## Layout primitives

- Use rows and columns for horizontal and vertical alignment.
- Use responsive containers to adapt to sidebar widths.
- Use spacing utilities for consistent padding and margins.

## Component composition

- Build complex panels by combining form groups, button groups, and content cards.
- Keep components small and reusable: a row with label + control, a collapsible section, a settings block.
- Prefer semantic structure over absolute positioning.

## Consistency guidelines

- Use consistent spacing between elements for readability.
- Keep buttons grouped near the controls they affect.
- Use headings and section labels to break large panels into digestible areas.

For layout examples and CSS conventions, see `docs/LAYOUT_COMPONENT_GUIDE.md`.
