This concept provides an index of the example workflows included with Sage Utils and how to use them as templates.

## Included examples

- `Ace 1.5 Example` — stable model prompt workflow.
- `Chroma Example` — vivid color styling and prompt composition.
- `Flux Dev Example` — development and debugging workflow.
- `LLM Example` — demonstrates LLM integration and prompt handoff.
- `Lumina 2 Example` — model configuration for Lumina 2.
- `NetaYume Lumina Example` — variant workflow for NetaYume Lumina.
- `NoobXL Example` — beginner-friendly model and prompt setup.
- `Pony Example` — pony-themed workflow with custom prompts.
- `Pony Upscaling Example` — includes an upscaling step.
- `Z-Image Example` — example for Z-Image workflow.

## How to use these examples

1. Open ComfyUI.
2. Choose Browse Templates.
3. Select the `comfyui_sageutils` package.
4. Import the example workflow that best matches your task.
5. Review node connections, prompt construction, and metadata outputs.

## Why this matters

Example workflows are intended to help users:
- learn the structure of Sage Utils graphs,
- discover how metadata nodes are used,
- and verify provider and model loading behavior.

## Notes

Each example consists of a `.json` workflow file and a sample output image.
Use the examples as starting points rather than finished production workflows.
