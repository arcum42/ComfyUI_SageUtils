This concept describes the example workflow gallery included with Sage Utils and how to use the provided templates.

## Gallery contents

Example workflows are stored in `example_workflows/` as JSON workflow files paired with sample images.

### Included examples
- `Ace 1.5 Example` — workflow for a stable Ace 1.5 model setup.
- `Chroma Example` — prompt and LoRA configuration for vivid chromatic styling.
- `Flux Dev Example` — a development-oriented example for workflow testing.
- `LLM Example` — demonstrates integration between the LLM tab and prompt builder.
- `Lumina 2 Example` — example using Lumina 2-compatible prompt and model settings.
- `NetaYume Lumina Example` — variant workflow built for NetaYume Lumina.
- `NoobXL Example` — beginner-friendly model pipeline.
- `Pony Example` — showcases pony-themed prompt generation.
- `Pony Upscaling Example` — includes an upscaling step.
- `Z-Image Example` — sample workflow for a Z-Image model.

## How to use

1. Open ComfyUI.
2. Use **Browse Templates**.
3. Select `comfyui_sageutils` from the template source list.
4. Choose an example workflow to import.

## Purpose

The example workflows are intended as starting points for:
- testing Sage Utils node behavior,
- exploring prompt builder and LLM interactions,
- validating metadata and model loader integrations,
- and learning by example.
